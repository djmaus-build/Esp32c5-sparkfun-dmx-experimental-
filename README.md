# Esp32c5-sparkfun-dmx-experimental-
Working esp32c5( 4MB ) with 1 artnet universe to dmx 512 output and a max485 driver output

Build with Arduino IDE 2.3.8 and the sparkfun DMX library 
working output pins TX-11 / Rx-12 / EN-23 
set onboard RGB-Led ( pin 27 ) as status led 

after upload the Wifi manager makes a hotspot ( 2,4Ghz ) to set your own wifi connection 

Perfect for DMX lightning and Movingheads on the almost free 5Ghz band
